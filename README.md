# WeekTrackerBot

Telegram-бот для фиксации рабочих достижений. Напоминает пользователю записать итоги периода (день, неделя, месяц) и сохраняет ответы в базе данных. Поддерживает несколько пользователей с индивидуальными расписаниями.

## Текущие возможности

- Регистрация пользователя через `/start`.
- Настройка расписания напоминаний:
  - Раз в день, в заданное время.
  - Раз в неделю, в выбранный день и время.
  - Раз в месяц, в указанное число и время.
- Автоматические напоминания с повторной проверкой через час, если запись не сделана.
- Сохранение текстовых записей с привязкой к периоду (день/неделя/месяц) в SQLite.
- Восстановление активных напоминаний после перезапуска бота.
- Отладочные команды `/time` и `/jobs` для контроля времени и очереди задач.

## Планы по доработке

- Интеграция LLM (локальной или облачной) для автоматической генерации ежемесячного саммари по команде `/summary`.
- Возможность редактирования и удаления прошлых записей.
- Экспорт записей за выбранный период.
- Поддержка индивидуальных часовых поясов для разных пользователей.
- Веб-интерфейс для просмотра статистики.

## Технологии

- Python 3.10+
- python-telegram-bot (с модулем job-queue)
- aiosqlite
- pytz (для часовых поясов)
